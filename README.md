# Juego de Triqui en Python

Este es un sencillo juego de triqui (tic-tac-toe) desarrollado en Python utilizando la biblioteca Pygame para la interfaz gráfica.

---

## Descripción

## El juego permite a dos jugadores alternar turnos para colocar sus símbolos ("X" y "O") en un tablero de 3x3.

## Características

- Interfaz gráfica simple y clara con imágenes para el fondo, la "X" y el "O".

- Cambio automático de turno entre los dos jugadores.

- Control del clic del mouse para colocar las fichas.

---

## Requisitos

- Python 3.x
- Pygame

Puedes instalar Pygame usando pip:

```bash
pip install pygame
```
